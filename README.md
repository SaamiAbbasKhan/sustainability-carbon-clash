# 🌍 Carbon Clash
`Made for 'Agents League Hackathon'`

### *The Planet Changes When Habits Change.*

Carbon Clash is a behavioral sustainability platform that combines **gamification**, **environmental simulations**, **AI-powered coaching**, **progression systems**, **virtual ecosystems**, and **social recognition** to transform sustainability from a passive concept into an engaging daily habit.

Carbon Clash is currently a **Prototype / MVP (Minimum Viable Product)**.

* Authentication screens and onboarding flows are implemented; full backend authentication and authorization are still being integrated.
* Certain sustainability calculations, progression systems, and environmental simulations are simplified for demonstration purposes.

# 🌎 Overview

Climate change is one of the defining challenges of our generation.

Every day, billions of small decisions collectively influence the future of our planet:

* The way we travel
* The products we buy
* The energy we consume
* The food we eat
* The waste we generate

While many people understand climate change exists, very few understand how their daily habits contribute to it or how small behavioral changes can collectively create meaningful impact.

Unfortunately, most sustainability applications struggle to retain users. They often focus heavily on information while overlooking motivation. Users are presented with carbon statistics, emissions reports, and educational material, but very little incentive exists for them to return the next day.

As a result, awareness increases while engagement decreases.

**Carbon Clash was created to address this challenge.**

---

# 🎯 Our Mission

Carbon Clash does not attempt to solve climate change through awareness alone.

Instead, it leverages:

* Gamification
* Behavioral Psychology
* AI-Driven Guidance
* Environmental Simulations
* Progression Systems
* Virtual Ecosystems
* Social Recognition

to transform sustainability from a passive concept into an engaging daily habit.

The objective is not simply to educate users.

The objective is to help users internalize sustainable thinking through repeated interaction, positive reinforcement, achievement systems, personalized guidance, and long-term engagement.

> Sustainability should not feel like homework.
> Sustainability should feel like progress.

Carbon Clash uses:

# ⚡ Eco Power

Eco Power is Carbon Clash's proprietary sustainability progression metric.

Rather than overwhelming users with technical carbon accounting terminology, Carbon Clash translates sustainability performance into a simple, understandable score.

## Why Eco Power Exists

* Makes sustainability easy to understand
* Encourages progress and comparison
* Turns complex environmental data into an engaging game mechanic

## How Eco Power Is Calculated

Eco Power is influenced by:

* Transportation habits
* Energy consumption
* Lifestyle decisions
* Sustainable actions
* Long-term consistency

### Positive Contributors

✅ Walking
✅ Cycling
✅ Public transportation
✅ Energy-saving habits
✅ Waste reduction
✅ Mission completion
✅ Daily streaks

> Eco Power is intentionally designed as an engagement-focused sustainability indicator rather than a strict scientific carbon accounting metric.

The platform analyzes:

to personalize which sustainability insights, challenges, and recommendations are presented.

This transforms static environmental information into contextual sustainability intelligence.

# 🚀 Quick Start

### Prerequisites

Make sure **Node.js** is installed on your system.

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

The application will start locally and be available in your browser.


That's it! 🎉 Carbon Clash should now be running locally.
